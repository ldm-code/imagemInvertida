# Imagem  Invertida

Um aplicativo web simples para estudo de js que permite ao usuário carregar uma imagem e girá-la suavemente em incrementos de 90° ao clicar num botao. A hora atual também é exibida na tela,
permanecendo fixa enquanto a imagem de fundo gira.

---

## Funcionalidades

- Carregar qualquer imagem do computador como fundo.
- Girar a imagem em 90°, 180°, 270° e 360° clicando no botão “Inverter”.
- Relógio em tempo real atualizado a cada segundo.
- Interface limpa, mantendo hora e controles fixos enquanto o fundo gira.
- Animação suave na rotação da imagem.

---

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript 

---

## Como usar

1. Abra o arquivo `imagem.html` no navegador.
2. Clique em **Escolher arquivo** e selecione uma imagem do seu computador.
3. Clique no botão **Inverter** para girar a imagem em incrementos de 90°.
4. Observe que a hora permanece centralizada e fixa.
